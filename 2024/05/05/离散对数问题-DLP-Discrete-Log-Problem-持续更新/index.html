自己写一段小数尝试找规律：

复数$$y_i$$有明显的循环结构，但是其阶和 ppp 的关系不明确，瞎猜与 φ(p)=p2−1\varphi(p) = p^2 - 1φ(p)=p2−1 有关（？）

但是模长平方再模除的循环结构很明确，不完全归纳猜测其阶即为 p−1p-1p−1

简单证明一下：

记某复数 ccc 及其共轭为：

c=(a+bi)cˉ=(a−bi)c = (a+bi)\\ \bar{c} = (a-bi) c=(a+bi)cˉ=(a−bi)

则二者与模长平方的关系有：

ccˉ=cˉc=a2−b2i2=a2+b2c \bar{c} = \bar{c} c = a^2 - b^2 i^2 = a^2 + b^2 ccˉ=cˉc=a2−b2i2=a2+b2

取正整数 nnn 作为指数：

(ccˉ)n=(cˉc)n=cncˉn=cˉncn=(a2+b2)n(c\bar{c})^n = (\bar{c} c)^n =c^n \bar{c} ^ n = \bar{c} ^ n c^n = (a^2 + b^2)^n (ccˉ)n=(cˉc)n=cncˉn=cˉncn=(a2+b2)n

关键的一个关系：

(ccˉ)n=cncˉn(c\bar{c})^n =c^n \bar{c} ^ n (ccˉ)n=cncˉn

其中 (ccˉ),(cncˉn)∈Z(c\bar{c}),(c^n \bar{c}^n) \in \mathbb{Z}(ccˉ),(cncˉn)∈Z，就可以应用模除了
